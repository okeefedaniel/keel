{% comment %} Fleet switcher — compact-by-default, hover/focus expanded, with cross-highlighting between the chip row and the expanded product list. Default: a small row of product logo chips in the sidebar footer. Each chip shows that product's actual favicon SVG (loaded from keel/core/static/img/fleet/.svg, which ships with keel so any deployment has all 9 logos). Hover or keyboard focus: the full product list appears as a popover above the compact row. Hovering a chip at the bottom highlights the matching row in the list above in a distinct "preview" color — so as the mouse slides left↔right across chips, the preview highlight moves up↔down in the list. Colors: - .active — current product (the one you're on now) - .hovering — the product you'd go to if you click (separate class so both can show simultaneously) Cross-highlight driven by data-product="" on each chip + list item, and a tiny inline script. Reads FLEET_PRODUCTS and CURRENT_PRODUCT from template context (injected by keel.core.context_processors.fleet_context). Each product in FLEET_PRODUCTS should have: name, label, code, url, icon.
